Linux

1) Посмотреть где я: 
pwd

2) Создать папку:    
mkadir folder

3) Зайти в папку:    
cd folder

4) Создать 3 папки:
mkdir folder_1 folder_2 folder_3

5) Зайти в любую папку: 
cd folder_1

6) Создать 5 файлов (3 txt, 2json):
touch 1.txt 2.txt 3.txt 1.json 2.json

7) Создать 3 папки:
пункт 4)

8) Вывести список содержимого папки: 
ls -la

9) + Открыть любой txt файл:
nano 1.txt или vim 1.txt (insert)

10) + написать туда что-нибуть, любой текст:	
"Hello World"
"Ubuntu"
"Terminal"
"Team"

11) + сохранить и выйти:
nano (Ctrl+x and y and inter) vim (esc : wq)

12) Выйти из папки на уровень выше:
cd ../

13) переместить любые 2 файлы, которые вы создали, в любую другую папку:
mv 2.txt 3.txt /folder_1 (если файл далеко от нужной папки то указывается полный путь к примеру (2.txt 3.txt/user/folder/folder_1))

14) скопировать любые 2 файла, которые вы создали, в любую другую папку:
cp 1.txt 1.json /folder_2

15) Найти файл по имени:
find -name 1.json

16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает:
grep -i 'Ubuntu' 1.json

17) вывести несколько первых строк из текстового файла:
head -n 2 1.json

18) вывести несколько последних строк из текстового файла:
tail -n 4 1.json

19) посмотреть содержимое длинного файла (команда less) изучите как она работает:
less 1.json

20) вывести дату и время:
date
